SimpleImageSlider

SimpleImageSlider is an open source iOS library available via CocoaPods to create a scrollable gallery of images.

EmailPicker

EmailPicker is an open source iOS library available via CocoaPods made by me at Dockwa Engineering. It is a UIViewController to make it easy for a user to select a list of emails by typing or choosing from their contacts.

RGB Remote

RGB Remote is an open source app that enables control of RGB LED strip lights by communicating with a server hosted on a Raspberry Pi running LIRC to control an IR blaster. Fork and modify to suit your needs!
